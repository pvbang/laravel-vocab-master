## laravel vocab master api

### Install: (Run in Terminal)
```bash
composer install 	
# or: composer install --ignore-platform-reqs
# or: composer update 
# or: composer update --ignore-platform-reqs 
cp .env.example .env
php artisan key:generate
php artisan serve
```
